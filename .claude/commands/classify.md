# Classify Emails Command

Run the email classifier to process recent Gmail emails.

## Execution

```bash
email-classify
```

This will:
1. Learn your writing style from sent emails
2. Fetch and analyze recent inbox emails
3. Classify which need responses (with priority ranking)
4. Generate personalized draft replies
5. Save drafts to Gmail

## Options

The command runs interactively:
- You'll paste prompts to Claude Code
- Claude analyzes and returns JSON
- The tool creates Gmail drafts automatically
